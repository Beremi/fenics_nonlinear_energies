Curated figure build support for the canonical `docs/assets/` tree.

What stays here:
- tracked source data under `data/`
- small rebuild scripts for the published family figures
- `build_all.py` to regenerate all current doc figures

What does not stay here:
- raw benchmark campaigns
- public markdown pages
- one-off publication rerun logs

- `docs/assets/plaplace/`
