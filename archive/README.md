older result notes that are useful for reference but do not describe the
current default implementation or the current final benchmark campaign.

- `investigation_*.md` and `*_INVESTIGATION.md` — one-off diagnosis and design studies
- `final_HE_results_archive_2026-03-08.md` — previous HE final-report snapshot
